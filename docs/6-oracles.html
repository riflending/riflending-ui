Oracles

An oracle is a way for a blockchain or smart contract to interact with external data. Since DEFI usually requires the interaction with real life information (i.e. the current rBTC price), Oracles are a critical component in every serious project. In this section, the rLending Oracle subsystem is explained in detail.

A Price Oracle maintains the current exchange rate of each supported asset. These exchange rates are used to determine borrowing capacity and collateral requirements, and for all functions which require calculating the value equivalent of an account.

Architecture:

rLending's Oracle subsystem is comprised by a set of smart contracts that delegate the ability to set the value of assets to a committee which pools prices from the top exchanges. Each asset listed has an Adapter contract that proxies the proper calls to the oracles providing the price information.
